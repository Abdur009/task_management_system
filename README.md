# Task Management System

A full-stack **Task Management System** built using **React.js**, **Node.js (Express)**, and **MySQL**.  
This application allows users to create, update, delete, and track tasks with statuses and deadlines.

---

## 🚀 Features

- Create, read, update, and delete tasks (CRUD)
- View all tasks in a task list
- View detailed information for a single task
- Update task status (Pending, In Progress, Completed)
- Search tasks by title or description
- Filter tasks by status
- Responsive user interface
- RESTful API architecture
- Input validation and proper error handling

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS and Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL
